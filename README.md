# Colab Projects - Scrapers & Automation

This repository contains Google Colab notebooks and automation scripts for data scraping and integration.  
Projects include:
- **Email Scraper**: Connects to an email server (requires server, email, password) and saves results into a Google Sheet stored in Drive.
- **Additional Scrapers**: Future notebooks for different sources and workflows.

## Setup
1. Open the notebook in Colab.
2. Provide required configuration (server, email, password, Google Sheet name).
3. Ensure the Google Sheet is created in Drive and shared with the Colab runtime.
4. Run cells to start scraping.
